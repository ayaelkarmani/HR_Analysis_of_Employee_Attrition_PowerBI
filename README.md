# 📊 Employee Attrition Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a comprehensive **Employee Attrition Analysis** using **Power BI**, based on HR data from *Atlas Labs*. The goal is to identify key factors influencing employee turnover and provide actionable insights to improve retention.

The dashboard is divided into **4 main sections**:

- Overview  
- Demographics  
- Performance Tracker  
- Attrition Analysis  

---

## 🎯 Objectives

- Analyze overall attrition trends  
- Identify high-risk departments and roles  
- Understand the impact of factors such as:  
  - Overtime  
  - Travel frequency  
  - Tenure  
  - Job satisfaction  
- Track employee performance and satisfaction over time  

---

## 🛠️ Tools & Technologies

- **Power BI** – Data visualization & dashboarding  
- **Power Query** – Data cleaning and transformation  
- **DAX** – Measures and calculated columns  
- **Excel / CSV** – Data source  

---

## 📈 Key Findings

### 🔸 Attrition Drivers

- Overtime significantly increases attrition risk  
- Employees with frequent travel show higher churn  
- Early tenure (0–2 years) has the highest attrition  

### 🔸 Department Insights

- Sales roles have the highest attrition rates  
- Technology has the largest workforce but lower relative attrition  

### 🔸 Performance Trends

- Lower satisfaction scores correlate with higher attrition  
- Work-life balance is a strong predictor of retention  

---

## 💡 Recommendations

- Reduce overtime workload or provide compensation incentives  
- Improve onboarding and support for new employees  
- Enhance work-life balance policies  
- Monitor satisfaction metrics proactively  
- Focus retention strategies on high-risk roles (e.g., Sales)
